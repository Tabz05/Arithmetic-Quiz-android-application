# Arithmetic-Quiz-android-application
Arithmetic Quiz - An android application to test mental math skills.

Overview: 

 Arithmetic Quiz is an android application through which users can
 check their mental math skills by taking calculation quizzes.
 There are 2 modes of quiz: Normal mode and Arcade mode.
 Each mode comprises 3 difficulty levels: easy,medium & difficult.
 Regardless of the difficulty and mode,there are 4 options for
 every question and out of the 4 options only 1 option is the
 correct answer to the question.
 
❖ The features of this app are:-

1. In normal mode, the users enter the number of minutes and the
number of seconds for which they want to play and the users
select the difficulty level. A timer runs while the game is being
played. At the end of the time specified by the user, the user is
presented with the result,i.e,number of correct answers, number
of wrong answers,etc.

2. In arcade mode, the users select the difficulty level and then play
the game. A stopwatch runs while the game is being played. The
game ends when the user gives a wrong answer to a question.
At the end of the game, the user is presented with the result,i.e,
the time for which the game lasted,number of correct
answers,etc.

3. Both modes can be played in 3 difficulty levels: easy,medium &
difficult. Regardless of the difficulty level and mode,there are 4
options for every question and out of the 4 options only 1 option
is the correct answer to the question.

4. In easy mode the correct option is easy to guess, as difficulty
level increases, it becomes difficult to guess the correct answer.

5. Regardless of the mode and difficulty level, while a game is
being played, the back button of the android device is disabled
so that the game does not terminate when the users press the
back button.

6. Users can sign up with email verification in order to get
their statistics (i.e. number of correct answers, number of
wrong answers, etc) saved when they play the game
while signed in.

7. While signing up, users cannot use an email id which is already
present in the database, thus,email id is unique for every user.

8. While signing up, users may choose to upload a profile picture or
they may choose to skip the uploading of profile picture. If a user
chooses to skip the uploading of profile picture, then a default
picture serves as the user’s profile picture. (However, the user
may upload a profile picture later, by editing his/her profile).

9. Users sign in by entering their email id and password.

10. When signed in, users can edit their profile picture.

11. The users can update their profile picture by selecting a
picture from the device.

12. Statistics are only saved if players play games while signed in.

13. When signed in, users can view their profile which consists of
username,email and profile picture.

14. When signed in,users can view their statistics.

15. When signed in, users may choose to reset their stats (i.e. set
all of their stats back to initial value, i.e set number of correct
answers,number of wrong answers, etc back to 0).

16. When users click on ‘reset stats’, a dialogue box appears
asking them to confirm whether or not they want to reset the
stats. If they click ‘Yes’, then their stats are reset. If they click
‘No’, then their stats are not reset.

17. Users can reset their password by receiving a password reset
email.

18. When signed in users may only play games or edit their profile
when they have an internet connection. Thus, for activities
requiring internet connection, it is checked whether or not users
have an active internet connection (i.e. wifi or mobile network). If
not, then they are told that internet access is not available.

19. Layout and look of pages (activites) change depending upon
the state of authentication (i.e.whether or not the user is signed
in).

20. Responsive and appealing layouts.

❖ Technologies used:-

➢ Programming Language: JAVA
➢ Integrated development environment (IDE) : Android Studio
➢ Database: Firebase Realtime-database & Firebase Storag
